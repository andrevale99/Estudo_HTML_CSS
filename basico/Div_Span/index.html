Bem vindo ao site com DIV's e SPAN's

Para realizar o agrupamento dos conteúdos deste site mixuruca, foram utilizados os elementos block (div) e inline (span).

"O div é um elemento HTML que tem a função de agrupar outros elementos. Além disso, o div é um componente block, ou seja, ocupa toda a área disponível no sentido horizontal. Para adicionar um elemento div no documento, basta utilizar a tag (div)." (IMD, Aula 08 - Elementos Block e Inline)

"O span, assim como a div, é um elemento HTML que tem a função de agrupar outros elementos. Diferentemente da div, o span é um componente inline, ou seja, ocupa apenas a área necessária para mostrar seu conteúdo. Para adicionar um elemento span no documento, basta utilizar a tag (span)." (IMD, Aula 08 - Elementos Block e Inline)

SOMENTE O CONTEÚDO

Podemos mudar o tamanho do span utilizando o elemento CSS display com o atributo inline-block. Esse atributo irá ocupar a área somente do conteúdo, mas o tamanho da caixa poderá ser modificado com os atributos width e/ou height
